Credit Card Churn


Hello! My name is Oleg! If you got to this page, then you are interested in my experience, so let me tell you what is going on here.

I created this app with FLASK-RESTful API to visually present the work I did on the Credit Card customers kaggle dataset. The problem is trivial - you need to build a model that can correctly classify whether the user is satisfied with banking services, and therefore whether he will remain a client in the future. Since we have highly structured (tabular) data, an effective solution was to use a model based on decision trees. LGBM - fast efficient boosting tree-based model, that has shown itself to be excellent among other candidates. So after creating a complete data preprocessing pipeline and model tuning, you can observe my result. Or just veiw source code.
